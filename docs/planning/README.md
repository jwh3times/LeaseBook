# Milestone retrospectives (M0–M7)

These are the **point-in-time retrospectives** for milestones M0–M7, published from the private
planning archive with confidential material redacted. Each records what its milestone delivered,
the integration-gate evidence, deviations from the plan, known limitations, and what the next
milestone had to absorb — **as of the date the milestone closed**.

They are historical records, not living documentation: the ADRs (`docs/adr/`) and the code
supersede anything stated here, and facts that were later corrected are deliberately left as
written. The corresponding milestone *plans* (`private/planning/m{N}_plan.md`) remain private,
as does `private/TODO.md` — paths under `private/` (and other gitignored working artifacts,
e.g. `.superpowers/`) are absent from a public clone, so citations to them in these documents
will not resolve here.
