# Superpowers specs & plans (point-in-time)

The documents under `plans/` and `specs/` are **design- and planning-session artifacts**, kept as
historical records of how each milestone's design was reached. They are accurate only as of their
stated dates: the ADRs ([`../adr/`](../adr/README.md)) and the code supersede them, and they are
deliberately not updated when reality moves on. Some cite `private/…` paths that are gitignored
and absent from a public clone. For the same reason this tree is excluded from the docs
link-check CI (`.github/workflows/docs-links.yml`) — a link that rots here is history, not a bug.
